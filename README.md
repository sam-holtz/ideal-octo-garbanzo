# Smart Banking Landing Page

A modern, responsive landing page for a smart banking platform designed for freelancers. Built with React and styled-components.

## Features

- **Modern Design** — Clean UI with gradient accents, smooth animations, and responsive layout
- **Smooth Scrolling** — Navigate between sections with animated scroll
- **Responsive** — Optimized for desktop, tablet, and mobile devices
- **Accessible** — Semantic HTML and keyboard-friendly navigation

## Tech Stack

- React 18
- Styled Components
- React Scroll
- AOS (Animate On Scroll)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Run tests
npm test
```

### Project Structure

```
src/
├── components/     # Reusable UI components
├── Sections/       # Page sections (Header, Pricing, FAQ, etc.)
├── assets/         # Images and icons
├── styles/         # Layout and global styles
└── App.js          # Main application
```
